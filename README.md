## Mac installation

Get the latest [Docker mac installer](https://docs.docker.com/docker-for-mac/).
***Docker machine is not used, as we use xhyve instead of virtualbox with this installer***


## Initial install
From root directory run:
```sh
bash scripts/init.sh
```

To build:
```sh
bash build.sh
```

To start:
```sh
bash start.sh
```
